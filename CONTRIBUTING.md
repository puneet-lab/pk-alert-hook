# Contributing

Thanks for your interest in **pk-alert-hook**!

This is a **personal project** maintained as a portfolio showcase. Contributions (pull requests, issues, feature requests) are **not accepted** at this time.

You're welcome to:

- **Use** the library in your own projects
- **Fork** and modify it for your own needs
- **Reference** the code for learning

If you find a critical bug, feel free to open an issue — but there's no guarantee it will be addressed.
